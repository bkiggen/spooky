## Installation Instructions


* *Clone this repository:*
  * In terminal, enter the following:
```
$ cd ~/desktop
$ git clone https://github.com/bkiggen/spooky
$ cd spooky
```
* *Install project dependencies*
  * In terminal, enter
```
$ npm install
```

* Open in browser
